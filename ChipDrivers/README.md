# ChipDrivers 层说明

`ChipDrivers/` 保存外部芯片或协议解析的可复用驱动。这里关心寄存器、协议字节、校验和数据转换，但不绑定本板引脚、片选、复位脚或具体外设实例。

## 目录

- `Inc/`：芯片驱动公开接口。
- `Src/`：芯片寄存器/协议实现。

## 当前内容

- `QMI8658`：IMU 寄存器驱动。
- `QMC6309`：磁力计寄存器驱动。
- `LT8920`：无线收发芯片驱动。
- `KCT8206`：射频前端控制。
- `gnss_nmea`：GPS NMEA0183 解析器。

## 边界

- 允许依赖 `McuAbstraction/` 的总线读写能力。
- 不允许写死板级引脚和设备实例。
- 不允许回调 App 或 BoardDevices。
